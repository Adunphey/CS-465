# CS-465
Full Stack Development I

As part of this project, a full stack web application was built that can handle both customer-facing and internal tasks. A MongoDB NoSQL database, an Express and Node.js server, and a single-page application (SPA) frontend make up the game. In the last version, security features were added to protect the master login and make sure that registration was safe.
From the point of view of design, this project showed a number of different ways to develop the front end. Express HTML was used for views that were produced by the server, and JavaScript was used to add dynamic features. The SPA made the user experience more current by changing content automatically without having to restart the whole page. This also made the site faster and easier to manage. The server database was chosen to be MongoDB because it is flexible, scalable, and works well with JavaScript-based apps and JSON-formatted data.
JSON is not the same as JavaScript because it is a lightweight, language-independent data file that is only used to send data. JSON is the way for the client and server of this app to talk to each other. Throughout the project, refactoring code made it more efficient and easier to read. Using reusable UI components cut down on duplicate work, improved consistency, and made the app easier to manage and add to.
During testing, different HTTP methods like GET, POST, PUT, and DELETE were used to make sure that API sites were working correctly. Security features added more complexity, which meant that identification, protected routes, and access control had to be tested. It became even more clear how important it is to know how routes, methods, and security levels work together in a full stack application.
This class improved my full stack development skills and taught me more about how current web apps are planned, built, and kept safe. Front-end coding, back-end APIs, database integration, and authentication skills I've learned have made me a more marketable option and given me a good portfolio item for future technical jobs.
